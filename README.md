\# 🛡 Intrusion Detection System using YOLO + SVM



\## 📌 Overview

This project implements a real-time facial recognition-based intrusion detection system.



The system detects faces using YOLO and classifies them using a Support Vector Machine (SVM) model to identify authorized and unauthorized individuals.



It is designed for surveillance and security monitoring applications.



---



\## 🚀 Features

\- Real-time face detection

\- Face recognition and classification

\- Unauthorized access detection

\- Modular Python-based architecture

\- Easy to extend and scale



---



\## 🛠 Tech Stack

\- Python

\- OpenCV

\- YOLO (Ultralytics)

\- Scikit-learn (SVM)

\- NumPy



---



\## 📂 Project Structure



```

app.py          → Main application entry point  

model.py        → Model loading \& prediction logic  

convert.py      → Preprocessing utilities  

storage.py      → Data handling  

yolo.py         → Detection pipeline  

data.yaml       → Dataset configuration  

requirements.txt → Project dependencies  

```



---



\## ⚙️ Installation



Clone the repository:



```bash

git clone https://github.com/sharmila-udaiyar/intrusion-detection-system-yolo.git

cd intrusion-detection-system-yolo

```



Install dependencies:



```bash

pip install -r requirements.txt

```



---



\## ▶️ Run the Project



```bash

python app.py

```
